如何使用Markdown编辑文档？

1 标题

这是最为常用的格式，在平时常用的的文本编辑器中大多是这样实现的：输入文本、选中文本、设置标题格式。

而在 Markdown 中，你只需要在文本前面加上 # 即可，同理、你还可以增加二级标题、三级标题、四级标题、五级标题和六级标题，总共六级，只需要增加 # 即可，标题字号相应降低。例如：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

注：# 和「一级标题」之间建议保留一个字符的空格，这是最标准的 Markdown 写法。

你可以你的编辑框中尝试输入这六级标题，发表文章后效果如下：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2 列表

列表格式也很常用，在 Markdown 中，你只需要在文字前面加上 - 或 + 或 * 就可以了，例如：

- 文本1
- 文本2
- 文本3
+ 文本1
+ 文本2
+ 文本3
* 文本1
* 文本2
* 文本3

你可以你的编辑框中尝试输入这上面任一种无序列表，发表文章后效果如下：

  • 文本1
  • 文本2
  • 文本3

如果你希望有序列表，也可以在文字前面加上 1. 2. 3. 就可以了，例如：

1. 文本1
2. 文本2
3. 文本3

注：-、+、*、1.和文本之间要保留一个字符的空格。

你可以你的编辑框中尝试输入这上面的有序列表，发表文章后效果如下：

  1. 文本1
  2. 文本2
  3. 文本3

3 代码区块

Markdown 提供 <pre> 和 <code> 标签来把代码区块包起来。 <code> 标签不会保留代码中的空格和换行符，所以单行的代码常用此来包括。 <pre> 标签可以保留代码中的空格和换行符。例如：

<code>lambda x,y:x+y</code>

会被转换成：lambda x,y:x+y

而使用<pre> 如下：

<pre>def plus(a,b):
	return a+b</pre>

会被转换成：

def plus(a,b):
	return a+b

4 插入链接

在 Markdown 中，插入链接不需要其他按钮，你只需要使用 [显示文本](链接地址) 这样的语法即可，例如：

[USTC Hacker](http://www.ustchacker.com)
会被转换成： USTC Hacker

5 插入图片

在 Markdown 中，插入图片不需要其他按钮，你只需要使用 ![](图片链接地址) 这样的语法即可，例如：

![](http://www.ustchacker.com/static/pic/unix.jpg)
会被转换成：

注：插入图片的语法和链接的语法很像，只是前面多了一个 ！。在本站发表文章的页面下方有上传图片的按钮，上传完成后会显示一个链接，使用这个链接即可。

6 粗体和斜体

Markdown 的粗体和斜体也非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。例如：

 *我们*都爱用**Python**

最终显示的就是下文，其中「我们」是斜体，「Python」是粗体：

我们都爱用Python

7 分割线

你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。下面每种写法都可以建立分割线：

* * *
***
*******
- - -
